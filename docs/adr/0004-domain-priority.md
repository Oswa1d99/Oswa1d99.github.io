# Launch on oswa1d99.github.io and keep jaybaek.dev as future domain

Jay Baek.dev will launch at `https://oswa1d99.github.io`. The site name remains `Jay Baek.dev`.

The custom domain `jaybaek.dev` remains the future preference because it is the strongest identity for the personal technical platform. This requires purchasing or otherwise owning the domain and configuring DNS for GitHub Pages.

The Deployment Module owns the active `site`, `base`, canonical URL, asset path, and smoke checklist details.

This supersedes ADR 0002's earlier assumption that v1 would deploy without a custom domain.
